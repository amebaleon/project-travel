# 프론트엔드 개발 기획서 (v1.2)

## 1. 개요

AI 여행 추천 서비스의 사용자 인터페이스(UI) 초기 버전을 개발합니다. 사용자의 입력을 받는 '입력 화면'과 추천 결과를 보여주는 '결과 화면' 두 페이지만 구현합니다.

## 2. 페이지 구성

### 2.1. 입력 화면

- **설명:** 사용자가 여행 추천을 받기 위해 필요한 정보를 입력하는 페이지입니다.
- **주요 기능:**
    - 지역, 여행 기간, 나이, 성별, 관심사 입력 폼
    - 백엔드에 추천을 요청하는 버튼

### 2.2. 결과 화면

- **설명:** AI로부터 받은 여행 추천 일정을 보여주는 페이지입니다.
- **주요 기능:**
    - 추천 장소 목록을 **카드 형태**로 표시합니다.
    - 각 카드에는 장소의 **이름, 설명, 활동, 주소** 등의 정보가 포함됩니다. (위도/경도 제외)
    - 실시간 검증 결과도 간결하게 표시됩니다.
